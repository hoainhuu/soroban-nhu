# Title
Campus Meal Credit – A Stellar-Based Cafeteria Credit System

# Description
Campus Meal Credit is a simple decentralized application built on Stellar Soroban to manage cafeteria meal credits for students.

The purpose of this project is to apply blockchain technology to a real campus use case. In many schools, meal payments are still handled by cash or separate transfers, which can be inconvenient and difficult to track. This project proposes a simple credit-based system where a cafeteria administrator can issue meal credits to students, and students can later use those credits for food purchases.

By building this project, I wanted to demonstrate that Stellar can be used not only for finance and remittance, but also for small, fast, and low-cost digital payment systems in everyday student life.

# Features
- Contract initialization with a designated admin
- On-chain storage of cafeteria meal credits
- Admin-controlled minting of meal credits
- Credit deduction when meals are purchased
- Balance checking for each user
- Self-spending function for users
- Credit transfer between accounts
